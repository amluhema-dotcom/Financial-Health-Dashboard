# 💰 Financial Health Dashboard

## 📌 Project Overview
The Financial Health Dashboard is an interactive Power BI dashboard that analyzes business financial performance. It provides insights into sales, profit, cost, product performance, and country-wise revenue using KPI cards, charts, and slicers.

---

## 🎯 Objectives
- Monitor business financial performance.
- Analyze sales and profit trends.
- Compare product and country performance.
- Create an interactive financial reporting dashboard.

---

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Microsoft Financial Sample Dataset (Excel)
- Microsoft Excel

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total COGS
- Total Units Sold

### Charts
- Sales by Country (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Sales by Product (Column Chart)
- Profit by Segment (Pie Chart)
- Profit by Country (Bar Chart)

### Interactive Slicers
- Country
- Product
- Segment

---

## 📈 Key Insights
- Identify countries with the highest sales.
- Monitor monthly sales trends.
- Compare product performance.
- Analyze profit contribution by business segment.
- Evaluate country-wise profitability.

---
